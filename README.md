# AIME2 Experiments
<a href="https://doi.org/10.5281/zenodo.20130652"><img src="https://zenodo.org/badge/1236173904.svg" alt="DOI"></a>

This repository contains the executable notebooks used to reproduce the experiments reported in the manuscript:

**AIME2: Toward a Unified Algebraic Theory of Explainability via Approximate Inverse Operators**

## Files

- `aime2_experiments.ipynb`: synthetic experiments for fidelity, stability, equivariance, and conditioning.
- `aime2_realdata.ipynb`: real-data comparison using the Wisconsin Diagnostic Breast Cancer dataset.

## Usage

Install dependencies:

```bash
pip install -r requirements.txt
```
Then run the notebooks:

1. aime2_experiments.ipynb
2. aime2_realdata.ipynb
